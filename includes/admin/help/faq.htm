Answers to Common Questions

Does Leaf have support for post formats?

Yes, Leaf supports 8 post formats, aside, audio, gallery, image, link, quote, status, and video. When adding/editing a post select the post format you want to use from the "Formats" meta box.

What page templates does Leaf have?

Leaf has 3 page templates, blog, one column, and HTML sitemap

To use a page template add/edit a page and in the "Page Attributes" meta box, under "Template", select the template you want to use. The blog template will display your post, the one column template will be a wide page with no sidebar, and the HTML sitemap will make a html sitemap for your site.
